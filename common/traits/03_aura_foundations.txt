# Foundation traits for Aura Kings.
#
# Inspired by the Aura Clash Foundations (see docs/AURA_CLASH_GAME_GUIDE.md §10):
# in the source a Foundation is a {elemental color base} + {Zodiac animal symbol}
# (e.g. "Swirling Wolf"), granting a core + element mastery + a skill, and above
# all STABILITY — a cultivator with a Foundation never sees their cores break or
# their Advancement bottleneck, "good all the way to Ascension."
#
# Here the 12 Zodiac animals are the buildable Foundation identities, each mapped
# to a distinct CK3 benefit. A character gains exactly one, at awakening (chosen
# or inherited — see events/aura_events.txt aura_kings.0004). The elemental
# color-base layer (core lean + element mastery) will be added once the element
# system exists.
#
# Each foundation also reserves hooks to "circle into other aspects" later:
# elements, techniques, spirit beasts, sects, and dynasty inheritance.
# Icons are referenced but not yet supplied; they fall back gracefully.
